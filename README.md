# ISHI Unit Generator

グリッドベースのシェイプ生成ツール。10:14のアスペクト比を持つキャンバス上で、グリッドの交差点をクリックしてシェイプを作成し、SVGとしてエクスポートできます。

## 機能

- **グリッドシステム**: 10×14のマス目、11×15の交差点
- **スナップ機能**: マウス位置から最も近い交差点に自動スナップ
- **複数シェイプ**: 1つのキャンバスに複数の独立したシェイプを描画可能
- **Undo機能**:
  - 描画中: 直前のポイントを取り消し
  - 待機時: 最後に作成したシェイプを削除
- **Reset機能**: キャンバスを全消去
- **SVGエクスポート**: 完成したシェイプをSVGファイルとしてダウンロード

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 使い方

1. グリッドの交差点をクリックしてパスを作成します
2. 始点の交差点を再度クリックすると、パスが閉じてシェイプが完成します
3. 複数のシェイプを描くことができます
4. 「Export SVG」ボタンでSVGファイルをダウンロードできます
